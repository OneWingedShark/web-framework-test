Mntext

License: SCD Conditions (www.softdevelcoop.org)
Version of this document: 20040419

Mntext is a textual language for the description of attribute instances.

Syntax

An attribute instance description is a non-null sequence of items, separated by white space.

An item is either an individual or a block.

An individual is either a string or a number.

A block is an attribute instance description enclosed in parenthesis.

Strings

A string is a sequence of characters representing its value. If the sequence includes white space then it must be delimited with double quotes. (The delimiters are not included in the value.)

The double quotes, the trim character, and the escape character, are special characters.

Trim character

The trim character is the vertical bar '|'. It effect is removing, from the value, itself and all white space immediately to the left up to and excluding the end of line if any.

This is useful for writing multiline text, specially if indented e.g. 

trim_character_example
   "for I in 1 .. 10 loop
   |   Do_Something_With (I);
   |end loop;"

Note the value includes the end of line characters (whatever they are).

Escape character

The escape character is the backslash '\'. Its effect is excluding itself from the value and blocking the interpretation of the successor character as a special one.

Naturally this is useful for including special characters in the value, e.g. 

escape_character_example
   "The last word in this line is \"quoted\""

Numbers

A number is a valid Ada number image.

Sets

A set is a (possibly null) list of items delimited by brackets and separated by ','. 

set_example
   {item1, item2, item3}

Vectors

The construct [item1, ..., itemN] is syntactic sugar for {1 item1, ..., N itemN}.

Semantics

The meaning of an attribute instance description is an attribute instance, as follows.

Any two consecutive items A B represent a vertex Y such that A→Y, Y→B, as defined by the Mneson Conventions for the representation of attribute instances in the untyped graph.

A description consisting of a single item A represents the vertex A.

The representation of sets as also as defined by the Mneson Conventions.

Notes

The compilation of an Mntext description requires an open graph.

The compilation of an Mntext description returns a vertex that represents the described attribute instance.

The indentification of the subject or subjects of the top level attribute instance is outside Mntext scope. The tool Mnedit provides a LINK command that can serve for this.
